# DJANGO CRUD

##### instalasi
```
$ git clone https://github.com/keselyoleren/django-crud.git
```
```
$ cd djano-crud
```
```
$ pip install django==2.1
```
```
$ python manange.py migrate
```
```
$ python manage.py runserver
```
```
http://localhost:8000
```
